# ○○町会ホームページ

町会のデジタル化に伴うホームページの作成・LINE での発信等の整備、および保守・管理をしていくための備忘・TODO をまとめるリポジトリです。

## ページ構成

| ページ | ファイル | 概要 |
|--------|----------|------|
| トップページ | `src/index.html` | サイト全体の案内・最新情報 |
| 回覧板 | `src/circular.html` | デジタル回覧板（お知らせ・PDF 掲載） |
| 災害情報 | `src/disaster-info.html` | 防災意識向上・外部リンク集 |
| 災害時の行動 | `src/disaster-action.html` | 警戒レベル・町民の対応・役員の対応・安否確認方法・避難所一覧 |
| 町会行事 | `src/events.html` | 年間行事・カレンダー |

## ローカル開発環境

Docker を使ったローカルサーバーを利用します。

```bash
# 起動
make up

# 停止
make stop

# ログ確認
make logs
```

起動後、ブラウザで http://localhost:8080 にアクセスするとサイトを確認できます。
